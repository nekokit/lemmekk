# Lemmekk 让我康康

lemmekk 目标是一个跨平台的工具。主要功能为调用 `7z` 批量解压文件，同时集成简单的密钥管理功能。

lemmekk 还在开发中，目标特性如下：

- [x] 简单密码管理
  - [x] 列出密码
  - [x] 添加密码
  - [x] 删除密码
  - [ ] 导入导出
    - [ ] 普通文本
    - [ ] `解TMD压` 格式
- [ ] 解压模块
  - [ ] 排除指定扩展名
  - [ ] 识别压缩包
  - [ ] 识别分卷
  - [ ] 识别图种
  - [ ] 递归解压
  - [ ] 解压后操作
    - [ ] 移动
    - [ ] 删除
